NewGameFormController
controller in module components.game

Description

Controller of component newGameForm. It controls process of defining new game options (at the moment - selecting correct mapId). On submit, it sends chosen value of mapId to the parent.

Methods

  • $onInit()

    Inits selected property as first of availableMaps

  • submitForm()

    It calls parent's onSubmit function with selected as mapId.

    Example

    <form novalidate ng-submit="$ctrl.submitForm();">
  <button type="submit">
   Create the game!
  </button>
</form>

Properties

  • selected

    Holds selected mapId
