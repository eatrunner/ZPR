newGameForm
directive in module components.game

Description

Reusable component that collects options for game create process and submits selected values to the parent

Dependencies

  • NewGameFormController

Usage

as element:
<new-game-form
       available-maps="{Array<number>}"
       on-submit="{Function}"
       error="{string}">
</new-game-form>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
availableMapsArray<number>

OneWay - List of possible mapId values

onSubmitFunction

Callback - function, when user hits submit button

errorstring

Input - reject message from parent
