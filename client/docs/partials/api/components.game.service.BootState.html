BootState
service in module components.game

Description

First state of the game, invoked at the beginning.

  • It configures game screen settings and loading assets (sprites, sounds),
  • It shows 'GAME IS BOOTING' text during load process
  • When loading is complete, game is switching into ReadyState

  • _loadAssets()

    It loads all assets needed in game, all sounds and all sprites. At the end of loading, it will call _loadComplete method.

  • _loadComplete()

    Callback method from Phaser.Game.load. Will be run after assets loading process. It starts next game state: LoadState

  • _showLoader()

    Member function that draws booting screen with blinking text

  • create()
