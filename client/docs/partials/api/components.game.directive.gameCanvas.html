gameCanvas
directive in module components.game

Description

<game-canvas> is a directive in which Phaser game is running. Here, game states are created.

When this directive is going to be destroyed, it destroys also Phaser resources

Game is created with size [384, 320] pixels. Phaser ScaleManager takes care about scaling, so when the screen goes resized, gameCanvas will also.

Dependencies

  • GameService
  • BootState
  • LoadState
  • PlayState
  • ReadyState

Usage

as attribute
<ANY game-canvas>
   ...
</ANY>
